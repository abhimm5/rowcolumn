Introduction

Why Row Column?

Rows & Columns is a JavaScript layout framework built on CSS Grid line placement, giving you fine-grained control over child <div> elements. While many developers prefer crafting custom layouts, building complex grids can become a headache due to deep nesting and hard-to-manage markup.

Two dimensional

This framework is designed to simplify layout design with minimal effort. Unlike approaches that rely on numerous wrappers like <div class="row">, it embraces a fully two-dimensional model.

In version v1.2.0, you can write real JavaScript logic directly in HTML attributes without using eval or regex-based parsers. 

<div layout="Grid.col(30, 70)">
    <div>Sidebar</div>
    <div>Content</div>
</div>
Getting Started

Installation

Zero dependencies. 5KB minified.

1. NPM (Recommended): 

npm install rowscolumns

2. CDN (Vanilla HTML):

<script type="module">
  import { Engine } from 'https://unpkg.com/rowscolumns';
  Engine.init();
</script>
Basics

Calling function

Grid.col() & Grid.row()

The library extends Number.prototype. Grid is a constant for 100.

Parameters

  • .col( ...sizes ): Divides the space vertically (creates columns).
  • .row( ...sizes ): Divides the space horizontally (creates rows).

These functions take numbers (or variables) as arguments.

Example

You can call this function like the example Grid.col(60, 40). Here Grid is a constant for 100.

<div layout="Grid.col(60, 40)">

Nesting (Recursion)

To divide into three parts or complex grids, you simply nest the functions. For example: Grid.col(60, (40).row(20, 20)).

Native Features

Native Power

1. Native Math

Because v1.2.0 uses native execution, you can use math directly. No manual percentages.

<div layout="Grid.col(100/3, 100/3, 100/3)">
33.3%
33.3%
33.3%

2. Variables

The engine provides built-in variables: lg (61.8) and sm (38.2).

<div layout="Grid.col(lg, sm)">

3. Global Functions

You can define a layout function in your script and call it in HTML.

<script>
  window.myLayout = (gap) => Grid.col(50, 50).props({ gap });
</script>
<div layout="myLayout('20px')">

Customisation

Splitting

Rational Division

To divide a number into two equal parts, we use Grid.col(1/2, 1/2). To make three columns, we use Grid.col(1/3, 1/3, 1/3).

Easy Recursive Loops

To make 12 equal columns manually would be tedious. Because v1.2.0 uses native JS, you can generate arrays dynamically!

<script>
  window.twelveCols = Array(12).fill(1); 
</script>
<div layout="Grid.col(...twelveCols)">

Controlling Dimensions

The number Grid (or 100) defines the ratio weights, not the actual pixel size. To change dimensions, apply CSS using .props().

<div layout="Grid.col(50, 50).props({ width: '50%' })">
Utilities

Styling & Methods

.props({ styles })

Apply CSS to the grid container (like gap, padding, margins).

Grid.col(50, 50).props({ gap: '20px', padding: '10px' })

.childProps({ styles }, [indices])

Apply CSS to specific children. Indices are 1-based.

Grid.col(30, 30, 30).childProps({ background: 'salmon' }, [2])

.offset([indices])

Responsive

Breakpoints

Rowscolumns uses specific attributes for breakpoints.

  • layout (Mobile / Default)
  • layout-sm (≥ 576px)
  • layout-md (≥ 768px)
  • layout-lg (≥ 992px)
  • layout-xl (≥ 1200px)
<div 
  layout="Grid.col(100)"             // Mobile: Stacked
  layout-md="Grid.col(50, 50)"       // Tablet: 2 Cols
  layout-xl="Grid.col(33, 33, 33)"   // Desktop: 3 Cols
>
Frameworks

React & TypeScript

Rowscolumns ships with a dedicated React wrapper and full TypeScript definitions.

1. The Setup

import React from 'react';
import { Layout, Grid, lg, sm } from 'rowscolumns/react';

2. Native Syntax

Pass real JavaScript objects to the layout prop. No quotes needed.

export const App = () => {
  const sidebar = 250; 
  
  return (
    <Layout layout={Grid.col(sidebar, 'auto')} >
       <div>Fixed Sidebar</div>
       <div>Fluid Content</div>
    </Layout>
  );
};
